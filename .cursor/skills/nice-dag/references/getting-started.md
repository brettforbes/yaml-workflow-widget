# Getting started

Source: `docs/getting-started.md`

## Install Vue3

```bash
yarn add @ebay/nice-dag-vue3
# or
npm install @ebay/nice-dag-vue3
```

## Install React

```bash
yarn add @ebay/nice-dag-react
# or
npm install @ebay/nice-dag-react
```

## First diagram

Vue: follow [vue-readonly.md](vue-readonly.md) (tutorial `docs/tutorial-vue3/read-only-dag.md`).

React: see `docs/tutorial-react/read-only-dag.md` and [react-hooks.md](react-hooks.md).

## This monorepo

Packages live under `packages/`:

- `nice-dag-core`
- `nice-dag-vue3`
- `nice-dag-react`

See [contribution.md](contribution.md) for `pnpm install` / `pnpm build` / examples.
