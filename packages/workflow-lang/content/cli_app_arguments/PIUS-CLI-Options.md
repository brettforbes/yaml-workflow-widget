Pius discovers CIDR blocks and domains associated with an organization using multiple OSINT data sources.

Usage:
  pius [command]

Available Commands:
  completion  Generate the autocompletion script for the specified shell
  help        Help about any command
  list        List available plugins
  run         Discover assets for an organization

Flags:
  -h, --help      help for pius
  -v, --version   version for pius

Use "pius [command] --help" for more information about a command.
